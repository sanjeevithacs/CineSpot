# **App Name**: CBE ShowTime

## Core Features:

- Dummy User Authentication: Allow users to sign up and log in using a username, userId, and password. Data is stored locally or in a database.
- City Selection: Allow users to select a city from a list of cities in Tamil Nadu.
- Movie Listings: Display a list of Tamil movies available in the selected city, using movie posters as thumbnails.
- Theatre Selection: List theatres in the selected city that are showing the selected movie.
- Show Time Selection: Allow users to choose from available show times (e.g., 10:00 AM, 2:00 PM, 6:00 PM, 9:30 PM).
- Seat Selection and Booking: Display seats in a grid format, allow users to select seats, prevent double-booking by disabling booked seats, and update seats booked by a logged-in user, so that next user will not book the same seat the previous user booked.
- Dummy Payment and Booking Confirmation: Calculate the dummy payment amount, process a dummy payment, and display a booking confirmation to the user.

## Style Guidelines:

- Primary Color: Maroon
- Background Color: Light Grey (#F5F5F5)
- Accent Color: Teal Blue (#00897B)
- Body and headline font: 'Lora', a serif font that provides readability and elegance.
- Use classic and elegant icons for navigation and seat selection.
- Employ a clean and structured layout for movie listings and seat selection, ensuring a user-friendly experience.
- Use subtle fade-in and fade-out animations for loading and transitions.
